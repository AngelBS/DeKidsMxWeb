Bienvenidos a DeKidsMx

Hoy aprenderemos a crear una página web desde 0, conoceremos que es HTML5 y nos introduciremos al increíble mundo de CSS3 para darle vida a nuestros proyectos, para que se vean super pros :D


Comencemos:

1.- Crear una carpeta con el nombre que queramos.

2.- Dentro de la carpeta crear un archivo llamada "index.html".

3.- Crear la estructura básica de HTML5.

4.- Jugar con las etiquetas básicas de HTML.

5.- Usar etiquetas de parrafo y encabezados grandes para mostrar diferentes contenidos.

6.- Al terminar de agregar todas las etiquetas a nuestra pagina esperar continuar con CSS3

¿Que es CSS3?

Es un lenguaje de Hojas de Estilo en Cascada, que se utiliza para darle forma y color a nuestra pagina para que sea más atractiva y se vea super bonita :3


CSS3

1.- Crear una carpeta llamada "estilos"

2.- Dentro de la carpeta "estilos" crear un archivo llamado "estilo.css".

3.- Añadir los atributos que desees para obtener una página web colorida y llamativa

NO ES NECESARIO QUE LA TERMINES, LO QUE NOS INTERESA ES QUE TE DIVIERTAS Y APRENDAS
