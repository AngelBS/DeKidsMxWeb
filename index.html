Bienvenidos a DeKidsMx

Hoy aprenderemos a crear una página web desde 0, conoceremos que es HTML5 y nos introduciremos al increíble mundo de CSS3 para darle vida a nuestros proyectos, para que se vean super pros :D


Comencemos:

1.- Crear una carpeta con el nombre que queramos.

2.- Dentro de la carpeta crear un archivo llamada "index.html".

3.- Crear la estructura básica de HTML5.

4.- Crear una carpeta con el nombre "estilos".

5.- Dentro de la carpeta "estilos" crear un archivo llamado "estilo.css".

6.- Añadir los atributos que desees para obtener una página web colorida y llamativa.

¿Que es CSS3?

Es un lenguaje de Hojas de Estilo en Cascada, que se utiliza para darle forma y color a una página web en HTML
